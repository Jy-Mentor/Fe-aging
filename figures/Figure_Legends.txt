METHODS SUMMARY
---------------
Data processing and visualization were performed using R (version 4.3.3) with
the following packages: ggplot2 (3.5.1), patchwork, ggrepel, ggsci, viridis,
dplyr, and readr. Single-cell analysis was conducted using Seurat (v5).
Ferroaging scores were computed using AddModuleScore. Differential expression
analysis was performed using limma with Benjamini-Hochberg multiple testing
correction. LASSO logistic regression was implemented via glmnet with 10-fold
cross-validation. GNN models (SAGE, HGT, SimpleHGN) were trained using PyTorch
Geometric. All statistical tests were two-sided unless otherwise specified.

METHODS SUMMARY (Supplementary)
-------------------------------
SCISSOR analysis was performed following Sun et al. (Nature Biotechnology,
2021; PMID: 34764492). Bulk RNA-seq data (GSE61616) was RMA-normalized and
ortholog-mapped (rat -> human -> mouse via babelgene/HomoloGene). Single-cell
data (GSE233815) was processed using Seurat v5. The SCISSOR algorithm computes
a Pearson correlation matrix S between bulk samples and single cells across
shared genes, followed by sparse regression on S with the phenotype vector Y.
The reference implementation uses network-regularized sparse regression with a
shared nearest neighbor (SNN) graph Laplacian penalty (alpha = 0.05); in this
study, we employed an engineering approximation using LassoCV (L1-regularized
regression) combined with stability selection (50 bootstrap iterations,
selection frequency threshold = 0.3) to identify Scissor+ (beta > 0) and
Scissor- (beta < 0) cells. Cell-type enrichment was assessed by hypergeometric
test. This approximation preserves the core sparse-selection principle of
SCISSOR while improving computational stability; the network regularization
term was not included.

Cell-cell communication analysis was performed using the CellChatDB.mouse
ligand-receptor database (Jin et al., Nature Communications, 2021;
PMID: 33597522; 3,379 interactions across 229 signaling pathway families).
Following the CellChat law-of-mass-action model, communication probability
was computed as P = L * R, where L and R are the expression levels of ligand
and receptor in sender and receiver cell types, respectively. For multi-subunit
complexes, expression was computed as the arithmetic mean across subunits
(CellChat v2 'mean' aggregation mode). Cell-type average expression was
computed as the arithmetic mean. Note: this implementation is a simplified
version of the full CellChat algorithm. The reference CellChat pipeline uses
geometric mean for multi-subunit complexes after network propagation on
STRINGdb (which projects zero-expression values to non-zero via random walk);
without network propagation, geometric mean is too strict for sparse scRNA-seq
data. The full CellChat algorithm additionally includes (i) trimean ensemble
averaging [ EM = 0.5*Q2 + 0.25*(Q1+Q3) ], (ii) Hill function transformation
(Kh = 0.5), (iii) agonist/antagonist and co-stimulatory/co-inhibitory cofactor
modulation, and (iv) permutation testing for statistical significance
(M = 100 permutations, P < 0.05). Interactions with probability > 0.01 were
retained. Analysis was implemented in R using circlize (chord diagram),
ggplot2 (heatmap, bubble plot, pathway bar chart), and patchwork (multi-panel
composite).
