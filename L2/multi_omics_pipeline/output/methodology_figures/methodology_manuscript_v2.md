# 铁衰老多组学交叉验证：从脑衰老代谢组到脑缺血转录组的跨模态证据整合

## 1  过程

本研究采用多组学交叉验证策略整合两个独立公共数据集。代谢组学数据来自 Metabolomics Workbench ST001637（C57BL/6 小鼠 3、16、59 周龄 10 个脑区 LC-MS/MS，1,709 种代谢物），转录组学数据来自 GEO GSE233815（小鼠 MCAO 模型 Ctrl/1DPI/3DPI/7DPI 大脑皮层 snRNA-seq，7,414 个细胞核）。铁衰老基因集 FA-96 源自 Liu 等（2026）系统筛选的铁死亡-衰老双重功能基因。代谢组丰度经 log2 转换与 Z-score 标准化后，采用两两 Welch t 检验（Benjamini-Hochberg FDR<0.05）进行差异分析，针对脂质过氧化、多胺、抗氧化、TCA、NAD+ 五类核心代谢构建铁衰老代谢特征评分。转录组使用 UCell 算法计算每个细胞核的 FA-96 富集分数。

跨组学整合分两层：通路轴验证（构建 8 条基因-代谢物轴，将衰老比较 3w vs 59w 中代谢物变化方向与基因功能预期进行一致性检验，按匹配率分级 ≥70%/50-69%/<50% 为强/中等/弱）；KEGG 通路整合（通过 KEGG REST API 查询 96 基因小鼠同源及 37 代谢物化合物 ID，识别共享通路）。代谢组外部验证使用 ST002042（MCAO 大鼠脑脂质组）和 ST002080（铁死亡诱导剂 RSL3 处理）数据集进行交叉验证。

## 2  结果

ST001637 中 3w vs 59w 共鉴定 242 个显著差异代谢物。亚牛磺酸下降最显著（log2FC=-2.62, p=9.8×10⁻³²），鸟氨酸（-0.89）、N8-乙酰亚精胺（-0.75）、腐胺（-0.64）均随衰老显著下降（图 1C）。还原型谷胱甘肽下降（-0.20, p=0.036），Cys-GSH 二硫化物上升（+0.61, p=3.2×10⁻⁸），提示氧化还原平衡向氧化方向偏移。

8 条通路轴中 3 条达"中等"证据等级（表 1，图 1A）。SAT1-多胺轴最优（5/7=71% 匹配率），鸟氨酸、N8-乙酰亚精胺、腐胺均显著下降，与 SAT1 激活导致精胺/亚精胺乙酰化耗竭的理论预期一致。SLC1A5-谷氨酸/半胱氨酸轴 8/13=62%，谷胱甘肽（-0.39, p=0.002）和甲硫氨酸（-0.39）均下降，支持 System Xc⁻ 抑制导致 GSH 合成受损。KEAP1-NRF2 抗氧化轴 5/10=50%。ACSL4-脂质信号轴仅 4/20=20%，神经酰胺类物质随衰老下降，与预期方向相反。

KEGG 整合中 96 基因全部映射至小鼠同源（697 条基因-通路链接），33/37 代谢物映射至化合物 ID（490 条链接），识别出 70 个共享通路（图 1B）。跨组学得分最高者为化学致癌-活性氧通路（mmu05208，11 基因，12 分），其次为神经退行性病变（mmu05022，10 基因，11 分）。铁死亡通路（mmu04216）含 5 个 FA-96 基因（ACSL4、ALOX15、HMOX1、LPCAT3、TFRC），HIF-1 信号通路（mmu04066）含 7 个基因。

## 3  讨论

本研究首次从代谢物层面为铁衰老假说提供跨组学证据。SAT1-多胺轴证据最扎实：鸟氨酸（-0.89）、N8-乙酰亚精胺（-0.75）、腐胺（-0.64）的显著下降与 SAT1 介导的精胺/亚精胺乙酰化加速理论一致。Turchi 等（2024, PMID 38787371）和 Liu 等（2024, PMID 36941264）在脑缺血模型中均观察到 SAT1 上调，提示 SAT1 激活是多胺耗竭的上游驱动，可能形成"铁死亡-多胺耗竭-衰老加速"正反馈环路。

SLC1A5-谷氨酸/半胱氨酸轴结果与 Stockwell 等（2022, PMID 38442890）的铁死亡执行机制吻合：谷胱甘肽和甲硫氨酸下降提示 System Xc⁻ 功能抑制，与脑缺血后 SLC7A11 表达下调的文献报道一致（PMID 40375180, 40768899），支持 GSH/GSSG 氧化还原失衡是衰老与脑缺血共同的铁死亡易感基础。KEGG 整合显示铁死亡通路（mmu04216）与 HIF-1 通路（mmu04066）均被 FA-96 基因与代谢物共同覆盖，化学致癌-活性氧通路以 12 分居首，反映 ROS 稳态失衡在铁衰老中的核心地位。

本研究局限：ST001637 为正常衰老小鼠全脑匀浆而非脑缺血模型，缺乏细胞类型分辨率；部分脂质（LPC、12-HETE）无法在 KEGG 精确定位。未来研究可结合空间代谢组学与靶向脂质组学在细胞分辨率上进一步验证铁衰老代谢基础。
